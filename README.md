# SJMCL Docs v2

这是一个基于 VitePress 的文档项目。

## 本地启动

先安装依赖：

```bash
npm install
```

启动本地开发服务器：

```bash
npm run docs:dev
```

启动后在浏览器打开终端里显示的本地地址，通常是：

```text
http://localhost:5173
```

## 其他常用命令

构建静态站点：

```bash
npm run docs:build
```

本地预览构建结果：

```bash
npm run docs:preview
```

## 项目结构

```text
docs/                  文档内容
docs/.vitepress/       VitePress 配置
```
