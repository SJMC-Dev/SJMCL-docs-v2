# 用户社群

欢迎所有 Minecraft 玩家加入 SJMCL 的相关用户社群，与其他的用户围绕启动器相关的使用体验、功能等等角度展开讨论，也可以在群内直接提出对启动器方面的问题、建议或者反馈，开发人员在看到后会进行解答与跟进。

Minecraft游戏内容相关话题也欢迎讨论交流，你可以畅所欲言，不过需要留意以下的群聊规范：

1. 保持和善、友好的态度，与其他群友围绕技术、体验、游玩等等方面展开交流；
2. 请勿在群内进行人身攻击、引战吵架、嘲讽、阴阳怪气等行为；
3. 请勿在群内发布黄色、涉政、与 Minecraft 无关的广告信息，以及任何不适宜的内容。公益服务器、整合包等可适当宣传；
4. 请各位群友务必注意个人隐私及信息安全。
