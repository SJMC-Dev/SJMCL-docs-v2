# SJMCL 界面与功能指南

欢迎使用 SJMCL (SJMC Launcher)！本指南将带您全面了解启动器的核心界面与功能模块，帮助您快速上手，享受高效、现代化的 Minecraft 启动与管理体验。

---

## 1. 顶部导航栏

该区域全局显示. 从左到右依次为五个界面模块: **启动界面**, **实例界面**, **账户界面**, **搜索与发现界面**和**设置界面**.

![顶部导航栏](/images/gui-topbar.png)

## 2. 启动界面 

启动界面是您进入游戏的第一站，在这里您可以快速选择账户、切换游戏实例并一键启动游戏。

*   **首次运行设定**：初次打开启动器时，会弹出以下界面，您可以阅读用户协议，并在左下角按需配置 **语言首选项**。
 ![首次运行](/images/gui-start-first-start.png)

*   **启动页面**：启动页面包含用户自定义背景，以及右下方的角色组件与启动游戏组件
 ![启动界面](/images/gui-start-overview.png)

*   **快捷切换与启动**：启动游戏前，点击“角色”或“游戏实例”右上角的**切换按钮**，即可在弹出的快捷悬浮窗中快速切换需要使用的账户或版本实例，点击**启动游戏**一键进入 Minecraft世界。
 ![角色切换](/images/gui-start-account-select.png)
 ![实例切换](/images/gui-start-game-version-select.png)

*   **游戏日志与崩溃分析**：
    *   **实时日志查阅**：在设置中开启“显示游戏日志”后，启动游戏时将自动弹出日志窗口。支持通过关键字筛选、按日志级别（FATAL, ERROR 等）过滤，并支持一键导出排错。
     ![游戏日志](/images/gui-start-game-log.png)
    *   **崩溃智能分析**：若游戏意外退出，SJMCL 会智能捕捉崩溃原因并展示错误分析窗口（包含 Java 版本、系统架构及崩溃详细内容），辅助您快速定位并解决问题。
     ![实例切换](/images/gui-start-game-error.png)

## 3. 实例界面

实例界面是您管理所有游戏版本、模组与存档的核心区域。SJMCL 采用了多目录、多实例隔离存储机制。
 ![实例界面总览](/images/gui-instance-overview.png)

### 全部实例

“全部实例”界面是对所有游戏目录下实例的统一展示，可以对各个实例进行快捷操作。
 ![全部实例](/images/gui-instance-all-instance.png)

#### 顶部栏

顶部栏从左到右依次为刷新、排序等：
* **刷新**：点击刷新当前界面
* **排序与筛选**：支持按游戏版本升序、降序与按实例名称排序
* **视图切换**：支持将各实例以网格视图或列表视图展示
* **添加与导入**：跳转到“添加与导入”界面
* **启动游戏**：启动当前选中的实例
 ![全部实例顶部栏](/images/gui-instance-topbar.png)

 #### 特定实例

 在网格视图中点击实例右上角的 **「…」** 可以选择跳转到所选实例所在的文件夹、实例详情页与删除实例
 ![特定实例](/images/gui-instance-all-instance-certain-one.png)

### 实例详情

点击侧边栏的实例，可以跳转到实例详情界面，该界面展示所选实例的多维信息，包括概览、模组、资源包、游戏内截图等
 ![实例详情](/images/gui-instance-sub-instance.png)

#### 顶部栏

顶部栏从左到右依次为实例名称、星标、打开文件夹、创建快捷启动方式等：
* **实例名称**：展示实例名称
* **星标**：星标当前实例后，当前实例会在侧边栏与“全部实例”界面中置顶（若排序未变化，可以点击“全部实例”的顶部栏中的“刷新”）
* **打开文件夹**：打开当前实例所在文件夹
* **创建快捷启动方式**：在桌面创建当前实例的快捷启动方式
* **导出整合包**：以整合包的形式导出当前实例
* **删除实例**：删除当前实例
* **启动实例**：启动当前实例
 ![实例详情顶部栏](/images/gui-instance-sub-instance-topbar.png)

### 添加与导入

点击 **「添加与导入」**，您可以：
* 一键安装全新的纯净版或模组版游戏实例
* 导入本地整合包，或直接从 CurseForge、Modrinth 等主流平台 **下载并导入整合包**
* 关联并管理本地已有游戏目录，无缝衔接旧版数据
* 下载各版本原版游戏服务端文件
 ![实例详情顶部栏](/images/gui-instance-new-instance.png)

## 5. 搜索与发现界面

SJMCL 不仅是开启游戏的钥匙，更是获取 Minecraft 深度资讯的视窗。

*   **聚合搜索**：在任意界面点击“搜索”，输入关键词即可一站式检索 **本地实例资源** 与 **线上游戏组件**。当启用“发现”后，可以用快捷键ctr+s快捷唤起。
 ![聚合搜索](/images/gui-search.png)
*   **发现资讯**：在“设置”界面的通用中开启“发现页”模式后，“搜索”栏将升级为 **「发现」**。
    *   您可以浏览设定数据源（如 SJMC 社团主页）的最新公告、新闻及干货分享；
    *   点击感兴趣的图文卡片可直达对应网页；
    *   支持自主管理并随时添加更多资讯数据源。
 ![发现资讯](/images/gui-find.png)
