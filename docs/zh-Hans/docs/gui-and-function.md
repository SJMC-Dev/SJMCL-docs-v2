# 界面与功能介绍

欢迎使用 SJMCL！本文档将带您全面了解启动器的界面与功能模块，帮助您快速上手，享受高效、现代化的 Minecraft 启动与管理体验。

## 顶部导航栏

该区域全局显示。从左到右依次为五个界面模块: `启动界面`、`实例界面`、`账户界面`、`发现界面` 和 `设置界面`。

![顶部导航栏](/images/gui-topbar.png)

## 启动界面 

启动界面是您进入游戏的第一站，在右下角可以快速选择账户、切换游戏实例并启动游戏。

![启动界面](/images/gui-start-overview.png)

* **快捷切换与启动**：启动游戏前，点击角色和实例卡片右上角的 `切换` 按钮，即可在弹出的快捷悬浮窗中快速切换需要使用的账户或版本实例，点击 `启动游戏` 一键进入 Minecraft 世界。

![切换账户与实例](/images/gui-start-switch.png)

## 实例界面

实例界面是您管理所有游戏版本、模组与存档的核心区域。SJMCL 采用了多目录、多实例隔离存储机制。

### 全部实例

**全部实例** 界面是对所有游戏目录下实例的统一展示，可以对各个实例进行快捷操作。

![实例界面总览](/images/gui-instance-overview.png)

实例页的顶部菜单从左到右依次为：

* **刷新**：点击刷新当前界面
* **排序与筛选**：支持按游戏版本升序、降序与按实例名称排序
* **视图切换**：支持将各实例以网格视图或列表视图展示
* **添加与导入**：跳转到 `添加与导入` 界面
* **启动游戏**：启动当前选中的实例

![全部实例顶部栏](/images/gui-instance-topbar.png)

### 实例详情

点击侧边栏的实例，可以跳转到实例详情界面，该界面展示所选实例的概览信息，你可以通过选项卡分别管理各类游戏资源。

![实例详情](/images/gui-instance-sub-instance.png)

实例详情页的顶部菜单从左到右依次为：

* **实例名称**：展示实例名称
* **星标**：星标当前实例后，当前实例会在侧边栏与 `全部实例` 界面中置顶（若排序未变化，可以点击 `全部实例` 顶部栏中的 `刷新`）
* **打开文件夹**：打开当前实例所在文件夹
* **创建快捷启动方式**：在桌面创建可以直接启动当前实例的快捷方式
* **导出整合包**：将当前实例导出为整合包
* **删除实例**：删除当前实例
* **启动实例**：启动当前实例

![实例详情顶部栏](/images/gui-instance-sub-instance-topbar.png)

### 添加与导入

点击左下角的 `添加与导入`，你可以：

* 安装全新的游戏实例，原版或包含模组加载器
* 导入本地整合包，或直接从 CurseForge、Modrinth 等主流平台 `下载并导入整合包`
* 关联并管理本地已有游戏目录，衔接其他启动器的实例数据
* 下载各版本原版游戏服务端文件

![添加与导入](/images/gui-instance-new-instance.png)

### 全局游戏设置

点击左侧菜单的 `全局游戏设置`，可以快捷跳转到 `设置` 界面中的全局游戏设置。

## 账户界面

SJMCL 拥有多用户验证系统，统一整合微软账户、离线账户与第三方认证（皮肤站）账户体系。可以分类查看并选择不同来源的角色，同样支持网格或列表形式直观展示。

![账户界面总览](/images/gui-account-overview.png)


## 搜索与发现界面

![发现界面总览](/images/gui-discover.png)

SJMCL 不仅是开启游戏的钥匙，更是获取 Minecraft 深度资讯的视窗。您可以在此使用聚合搜索、查看新闻与热门整合包。

* **聚合搜索**：输入关键词可集中检索 **本地账户、游戏资源** 与 **在线游戏资源**。可以用快捷键 `Ctrl`（macOS 为 `⌘`） + `S` 快捷唤起。
