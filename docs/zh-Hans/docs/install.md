# 安装 SJMCL

从本站[下载页](/downloads/)获取适合你平台的最新版本，也可以前往 [GitHub Releases](https://github.com/UNIkeEN/SJMCL/releases) 获取历史版本。

## 系统要求

| 平台 | 支持版本 |
| --- | --- |
| Windows | Windows 10 / 11（x86、x64、ARM64） |
| macOS | Apple Silicon 与 Intel |
| Linux | x64、ARM64 |

## Windows

提供安装版 (Setup) 与便携版 (Portable)，推荐使用**便携版**——下载后即可直接运行，无需安装。

| 格式 | 说明 |
| --- | --- |
| **便携版**（推荐） | 单文件，下载后双击即可运行，适合大多数用户 |
| 安装版 | 通过安装向导安装到系统中 |

可用架构：x64、x86、ARM64。

## macOS

提供 DMG 安装包与 App Bundle，推荐使用 **DMG**——打开后将应用拖入 Applications 文件夹即可。

| 格式 | 说明 |
| --- | --- |
| **DMG**（推荐） | 标准 macOS 安装方式，拖拽安装 |
| App Bundle | `.app.tar.gz` 压缩包，解压后使用 |

可用架构：Apple Silicon、Intel。

## Linux

提供多种分发格式，按需选择。

| 格式 | 说明 |
| --- | --- |
| AppImage | 通用格式，下载后赋予可执行权限即可运行 |
| DEB | 适用于 Debian / Ubuntu 系发行版 |
| RPM | 适用于 Fedora / openSUSE 等发行版 |
| 便携版 | 单文件可执行程序 |

可用架构：x64、ARM64。
