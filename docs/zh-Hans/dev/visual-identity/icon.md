# 徽标

徽标是 SJMC Launcher 视觉形象的核心。

在进行与启动器相关的内容设计时，建议您使用与官方一致的徽标进行创作，以保证形象的一致性。

## 徽标释义

SJMC Launcher 徽标由中央主图样、上海交通大学东大门剪影以及渐变正六边形背景构成。

主图样参考上海交通大学校徽中的书本、铁砧、齿轮等设计元素，结合别具 Minecraft 特色的苦力怕面部图案设计而成。东大门剪影更凸显出交大特色。

徽标总体体现开发团队以交大精神为底色，承载上海交通大学 Minecraft 社爱 MC 更爱交大的初心与对技术开发的热爱，以及在此之上希望为社区打造一款实用且富有活力的启动器的本心。
