# 开源协议

该软件依据 [GNU 通用公共许可证第 3 版（GPLv3）](https://github.com/UNIkeEN/SJMCL/blob/main/LICENSE) 进行分发。

根据 GPLv3 许可协议第 7 条的规定，当您分发本软件的修改版本时，除遵守 GPLv3 外，还须遵守以下 [附加条款](https://github.com/UNIkeEN/SJMCL/blob/main/LICENSE.EXTRA)：

1. 必须更换软件名称，禁止使用 SJMCL 或 SJMC Launcher；
2. 在您的仓库 README、分发网站或相关文档、软件的关于页面中，须明确标注您的程序基于 SJMCL，并注明原仓库链接。
3. 当对本软件的修改仅限于**增加**而非修改或删除预置认证服务器时，前述第 1 条限制不适用。在该情形下，您可继续使用原始的软件名称进行编译与分发。

另根据我们网站的用户协议，当您分发本软件的修改版本时，请仅向我们的信息统计服务器发送带前缀的版本号，除非你的修改满足上述第 3 条限制。
