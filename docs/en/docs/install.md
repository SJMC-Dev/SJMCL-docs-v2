# Install SJMCL

Download the latest version for your platform from the [Downloads page](/downloads/), or visit [GitHub Releases](https://github.com/UNIkeEN/SJMCL/releases) for older versions.

## System Requirements

| Platform | Supported |
| --- | --- |
| Windows | Windows 10 / 11 (x86, x64, ARM64) |
| macOS | Apple Silicon and Intel |
| Linux | x64, ARM64 |

## Windows

Available as Setup and Portable builds. **Portable is recommended** — just download and run, no installation needed.

| Format | Description |
| --- | --- |
| **Portable** (recommended) | Single executable, download and double-click to run |
| Setup | Installs via a setup wizard |

Available architectures: x64, x86, ARM64.

## macOS

Available as DMG installer and App Bundle. **DMG is recommended** — open the disk image and drag the app into your Applications folder.

| Format | Description |
| --- | --- |
| **DMG** (recommended) | Standard macOS drag-and-drop install |
| App Bundle | `.app.tar.gz` archive, extract and use |

Available architectures: Apple Silicon, Intel.

## Linux

Multiple distribution formats are available.

| Format | Description |
| --- | --- |
| AppImage | Universal format, make executable and run |
| DEB | For Debian / Ubuntu based distributions |
| RPM | For Fedora / openSUSE based distributions |
| Portable | Single executable binary |

Available architectures: x64, ARM64.
