# Introduction

SJMC Launcher is a modern, cross-platform Minecraft launcher built on the Tauri framework, independently developed by members of the Shanghai Jiao Tong University Minecraft Club.

Ready to get started? Head to [Install SJMCL](/en/downloads/).

> [!IMPORTANT]
> NOT AN OFFICIAL MINECRAFT SERVICE. NOT APPROVED BY OR ASSOCIATED WITH MOJANG OR MICROSOFT.

## Features

- **Cross Platform**: Supports Windows 10/11, macOS and Linux.
- **Efficient Instance Management**: Supports multiple game directories and instances, allowing the management of all instance resources such as saves, mods, resource packs, shaders, screenshots and settings in one place.
- **Convenient Resource Download**: Supports downloading game clients, mod loaders, various game resources and modpacks from CurseForge and Modrinth.
- **Multi-Account Support**: Built-in Microsoft login and third-party authentication server support, compatible with the OAuth login flow proposed by Yggdrasil Connect.
- **Deeplink Integration**: Integrates with external websites and tool collections, making it easier to launch instances from system deeplinks or desktop shortcuts.

> Note: some features may be limited by region, platform, or bundle type.
